# Dia 45 — 2026-08-25 (terça-feira)

### Semana 7 — Marcar compromissos / telefone

## 🎯 Objetivo do dia
Aprender frases para marcar horários e falar ao telefone.

## 📚 Vocabulário novo

| Neerlandês | Português |
|---|---|
| Kan ik een afspraak maken? | Posso marcar um horário/compromisso? |
| Met wie spreek ik? | Com quem eu falo? (ao telefone) |
| Een ogenblikje, alstublieft | Um momentinho, por favor |
| Tot volgende week! | Até semana que vem! |
| de afspraak | o compromisso/consulta |
| Ik bel terug | Eu ligo de volta |

## 📖 Gramática

Essas frases são muito úteis para situações práticas reais (marcar médico, cabeleireiro, etc.) — vale a pena praticar em voz alta como se estivesse realmente ao telefone.

## 💬 Frases modelo

- Kan ik een afspraak maken voor volgende week? — Posso marcar um horário para semana que vem?
- Met wie spreek ik? — Com quem eu falo?
- Ik bel u later terug. — Eu ligo para o senhor mais tarde.

## ✏️ Exercícios

1. Escreva um diálogo completo de ligar para marcar uma consulta médica (5-6 trocas de fala).
2. Grave-se encenando essa ligação telefônica.
3. Traduza: 'Posso marcar um horário para amanhã de manhã?'
