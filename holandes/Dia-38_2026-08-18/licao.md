# Dia 38 — 2026-08-18 (terça-feira)

### Semana 6 — Particípios irregulares comuns

## 🎯 Objetivo do dia
Memorizar os particípios irregulares mais usados no dia a dia.

## 📚 Vocabulário novo

| Neerlandês | Português |
|---|---|
| zijn → geweest | ser/estar → sido/estado |
| hebben → gehad | ter → tido |
| gaan → gegaan | ir → ido |
| doen → gedaan | fazer → feito |
| zien → gezien | ver → visto |
| eten → gegeten | comer → comido |
| drinken → gedronken | beber → bebido |
| komen → gekomen | vir → vindo |

## 📖 Gramática

Infelizmente esses verbos muito comuns não seguem a regra regular — precisam ser decorados como uma lista, igual os verbos irregulares do português (ex: 'ir → fui', não 'ir → ii').
Dica: são só 8 hoje, mas são os mais usados na fala do dia a dia — vale a pena criar flashcards físicos ou no celular para esses.

## 💬 Frases modelo

- Ik ben gisteren naar het park geweest. — Eu estive no parque ontem.
- Wat heb je gegeten? — O que você comeu?
- Wij hebben een film gezien. — Nós vimos um filme.

## ✏️ Exercícios

1. Crie flashcards (papel ou app) com os 8 particípios irregulares de hoje, infinitivo de um lado e particípio do outro.
2. Escreva 8 frases no passado, uma para cada verbo irregular.
3. Traduza: 'Eu bebi café.' / 'Ela foi ao mercado.' / 'Nós fizemos o trabalho de casa.'
