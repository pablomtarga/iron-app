# Dia 15 — 2026-07-26 (domingo)

### Semana 3 — Família

## 🎯 Objetivo do dia
Aprender vocabulário de membros da família.

## 📚 Vocabulário novo

| Neerlandês | Português |
|---|---|
| vader | pai |
| moeder | mãe |
| ouders | pais (ambos) |
| broer | irmão |
| zus | irmã |
| opa | avô |
| oma | avó |
| oom | tio |
| tante | tia |
| neef | primo/sobrinho |
| nicht | prima/sobrinha |
| zoon | filho |
| dochter | filha |
| man | marido/homem |
| vrouw | esposa/mulher |

## 📖 Gramática

Repare que 'neef' e 'nicht' significam tanto primo/prima quanto sobrinho/sobrinha — o contexto define. E 'man'/'vrouw' significam tanto 'homem/mulher' quanto 'marido/esposa'.
Todas essas palavras de família são 'de' (nenhuma é 'het') — adicione ao seu caderno de vocabulário.

## 💬 Frases modelo

- Dit is mijn vader. — Este é meu pai.
- Ik heb een broer en twee zussen. — Eu tenho um irmão e duas irmãs.
- Mijn ouders wonen in Brazilië. — Meus pais moram no Brasil.
