# Dia 16 — 2026-07-27 (segunda-feira)

### Semana 3 — Posse (mijn, jouw, zijn, haar...)

## 🎯 Objetivo do dia
Aprender os pronomes possessivos.

## 📚 Vocabulário novo

| Neerlandês | Português |
|---|---|
| mijn | meu/minha |
| jouw / je | teu/tua (informal) |
| uw | seu/sua (formal) |
| zijn | dele (seu/sua) |
| haar | dela (seu/sua) |
| ons / onze | nosso/nossa |
| jullie | vosso/de vocês |
| hun | deles/delas |

## 📖 Gramática

Os possessivos não mudam com o gênero do substantivo, EXCETO 'ons/onze':
- **ons** + palavra 'het' (ex: ons huis - nossa casa)
- **onze** + palavra 'de' (ex: onze auto - nosso carro)
Todos os outros (mijn, jouw, zijn, haar, jullie, hun) são sempre iguais, não importa de/het.

## 💬 Frases modelo

- Dit is mijn huis. — Esta é minha casa.
- Waar is jouw auto? — Onde está o teu carro?
- Ons huis is groot, onze auto is klein. — Nossa casa é grande, nosso carro é pequeno.
