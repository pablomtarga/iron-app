# Dia 41 — 2026-08-21 (sexta-feira)

### Semana 6 — Comparativo e superlativo

## 🎯 Objetivo do dia
Aprender a comparar coisas e usar o superlativo.

## 📚 Vocabulário novo

| Neerlandês | Português |
|---|---|
| adjetivo + er | comparativo (mais ___) |
| adjetivo + st | superlativo (o mais ___) |
| goed → beter → best | bom → melhor → o melhor |
| groot → groter → grootst | grande → maior → o maior |
| dan | do que (em comparações) |

## 📖 Gramática

Regra geral: **adjetivo + er** = comparativo ('groter' = maior), **de/het + adjetivo + st(e)** = superlativo ('de grootste' = o maior).
'Goed' (bom) é irregular, igual em português: goed → beter → best (bom → melhor → ótimo/o melhor).
Para comparar duas coisas, use **dan** (do que): 'Amsterdam is groter dan Utrecht.' (Amsterdã é maior que Utrecht.)

## 💬 Frases modelo

- Mijn broer is groter dan ik. — Meu irmão é maior/mais alto que eu.
- Dit is de beste koffie. — Este é o melhor café.
- Nederlands is makkelijker dan ik dacht. — Holandês é mais fácil do que eu pensava.

## ✏️ Exercícios

1. Compare 3 pares de coisas/pessoas usando comparativo + 'dan'.
2. Escreva 3 frases usando o superlativo (de/het + adjetivo + st).
3. Traduza: 'Este livro é melhor do que aquele.' / 'Ela é a mais inteligente da família.'
